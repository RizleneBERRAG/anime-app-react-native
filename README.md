Anime Explorer App

Objectif du projet

Créer une application mobile complète permettant d'explorer des animesen utilisant une API publique, avec les fonctionnalités suivantes :

* Navigation entre écrans
* Liste d'animes
* Scroll infini (pagination)
* Barre de recherche optimisée
* Écran de détail d'un anime

L'application utilise l'API Jikan (MyAnimeList).
